# Nexus AI chat integration

For every module / project, there should be a button that toggles on/off a small chat window (on the right side) that let's the user interact with the AI in the given projet. This is also where , if Claude is used, the /init command can be executed to let Claude do its thing.
It also allows for communication with other LLMs based on API key or locally run LLMs, either external (as set up in the settings) or from within the localai module.
