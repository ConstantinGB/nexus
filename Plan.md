# Module Uplifts (Cosmetic + Function) — Implementation Plan

## Context

We're uplifting the module usage, mostly cosmetically

---

## Start Screen

### Project Tiles

**Current State:** Shows the full directory name, including Module prefix, name and description, all left-bound. 

**Change:** We will change the appearance in the following ways: 

- All text ist centered
- The NAME of the project will be displayed most prominently, fat text, larger than the rest of the Text. The name will be displayed WITHOUT module prefix. 
- Beneath the name, smaller but still prominent, will be the module type. (Full name module type, not prefix, the prefixes are more for directory structure and to prevent overlap)
- Beneath both will be the description. 
- Tile size itself should remain the same. We have enough free space to work with. 
## Module / Project screen

### Editing

**Current State:** When set up, the name and description are written once and then unchanging. 

**Change:** (Global) Editing-Button (gear icon) that opens a screen which allows for editing both the project name and the project description.
